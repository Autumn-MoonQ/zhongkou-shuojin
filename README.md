# 众口铄金

> 热点整合·舆论分析·文科论述题深度解析

## 项目简介

立足新媒体时代信息传播特点与内容创作、文科学习的实际需求，打造集热点整合、舆论分析、文科论述题深度解析于一体的专业内容体系。

### 核心功能

- **热点整合**：聚焦当下全网热门话题进行集中梳理整合，打破平台信息壁垒
- **舆论分析**：针对全网热点事件展开客观中立的全维度分析
- **论述题解析**：深入剖析文科论述题的核心内涵、逻辑框架与作答思路
- **用户系统**：用户注册、登录、收藏、评论等交互功能

## 技术栈

- **前端**：Vue 3 + Nuxt 3 + Element Plus + Pinia
- **后端**：Nuxt Server API
- **数据库**：SQLite (better-sqlite3)
- **样式**：SCSS

## 快速开始

```bash
# 安装依赖
npm install

# 初始化数据库
npm run db:init

# 启动开发服务器
npm run dev
```

## 项目结构

```
众口铄金/
├── assets/              # 静态资源
│   └── styles/         # 全局样式
├── components/         # Vue组件
├── composables/        # 组合式函数
├── layouts/            # 布局组件
├── pages/              # 页面路由
│   ├── hotspots/       # 热点整合
│   ├── analysis/       # 舆论分析
│   ├── essays/         # 论述题解析
│   └── user/           # 用户中心
├── server/             # 服务端代码
│   ├── api/            # API接口
│   ├── db/             # 数据库
│   └── middleware/     # 中间件
├── stores/             # Pinia状态管理
└── types/              # TypeScript类型定义
```
